Step 1:

Click on CIS Deployment in the Jump Drive

 

Step 2:

Highlight the following:

·         Application Files

·         CISApp(File Folder)

·         CISApp(Click Once Application)

·         Setup

·         Sqlite-netFx40-setup-bundle-x86-2010

·         Sqliteodbc_w86

 

Step 3:

Copy the selected files from step 2 and paste them to your desktop.

 

Step 4:

Double-click on the sqliteodbc_w86 icon to star the SQLite Database install.

Step 5:

Click next. This installs the Database that you will be using on your machine.

 

Step 6:

Neither of these checkboxes need to be selected. Click the Install button to Install the program.

 

Step 7:

Double-click the sqlite-netFx40-setup-bundle icon to start the install.

 

Step 8:

Please make sure the above checkboxes are selected and click the Next button.

 

During the installation process a DOS window will pop up momentarily. This is the database installing itself on your machine.


 

Step 9:

Click the Setup icon on your desktop.

 

This process will check to see what programs will have to be installed on your machine in order for the application to work correctly. This may take a few minutes depending on which programs you will need. Any missing programs will be downloaded automatically.

 

Step 10:

Click on the Start Menu in Windows and select Control Panel.

Step 11:

Please click on System and Security.

Step 12:

Please select Administrative Tools.

 

Step 13:

Select Data Sources(ODBC).

 

Step 14:

On the right-hand side of this dialog box, click the Add button.

 

Step 15:

This will bring up the Create New Data Source Dialog box. Scroll down in the Name Textfield until you find the SQLite3 ODBC Driver option. Please select this by clicking the Finish button.

 

Step 16:

This will bring up the DSN Configuration dialog box. In the Data Source Name textbox please enter CISdb. This sets the name of the database.

 

Step 17:

Please click on the Browse button.

 

Step 18:

Once the Browse dialog box is open, select the CISApp file folder and click Open.

Step 19:

With the CISApp folder open, browse to and select the CISdb and click the Open button.

Step 20:

Click Install.
